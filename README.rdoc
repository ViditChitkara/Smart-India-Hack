== CPI Web Portal/ App Backend (Team Name: What the bug)

Ruby on rails backend for CPI android app.

== Installation Instruction

* Set the development environment for Ruby on rails.

*Commands:*

* `bundle install` --> To install libraries
* `rake db:migrate` --> To set database
* `rake db:seed` --> For test data
* `rails s` --> To start the server locally

Producution link: https://sih-cpi.herokuapp.com/
